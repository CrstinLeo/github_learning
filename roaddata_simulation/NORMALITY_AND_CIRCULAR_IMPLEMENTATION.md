实现了两种方法供选择：

**方法1：IQR（四分位数范围）法**
```
Q1 = 25th percentile
Q3 = 75th percentile  
IQR = Q3 - Q1
异常值阈值: [Q1 - 1.5×IQR, Q3 + 1.5×IQR]
```

**方法2：Hampel 法（基于中位数绝对偏差）**
```
Median = 中位数
MAD = 中位数绝对偏差
异常值判断: |x - Median| > threshold × 1.4826 × MAD
```

#### 2. **圆统计计算**（已完成）

**核心公式**：
```
弧度转换：rad = deg × π/180

sin均值：sin_mean = mean(sin(rad))
cos均值：cos_mean = mean(cos(rad))

圆均值：mean = atan2(sin_mean, cos_mean) → 转换回度数并映射到[0,360)

结果向量长度：R = √(sin_mean² + cos_mean²)  [范围: 0~1]

圆标准差：CircStd = √(-2 × ln(R))
```

**实际计算结果**：
- 所有 segment 的 R > 0.99（极高的方向一致性）
- 圆均值范围：0-360°（正确处理了循环边界）
- 圆标准差范围：0.09-0.16 rad

#### 3. **风向稳定性**（已完成）
基于 R 值的评估标准：
```
R > 0.9   → Very High (consistent direction) ✓✓✓
0.7 < R ≤ 0.9   → High ✓✓
0.5 < R ≤ 0.7   → Moderate ✓
R ≤ 0.5   → Low (dispersed) ✗
```

**实际结果**：
- **所有 20 个 segment 的 R > 0.99**
- **全部评级为 "Very High (consistent direction)"**
- 说明风向分布极度集中，几乎所有测量点都指向主导方向

## 使用建议

1. **风速分析**：
   - 优先使用清洁后的数据和 P90 指标
   - 关注异常值比例，高于 5% 需要检查数据质量
   - 参考 Shapiro-Wilk p-value 判断是否可用参数统计

2. **风向分析**：
   - 始终使用圆统计结果，不要用线性均值
   - 根据 R 值判断风向稳定性
   - R > 0.95 可视为高可靠度

3. **后续建模**：
   - 使用圆均值而非线性均值
   - 考虑风速的非正态性（必要时用非参数方法）
   - 风向可用圆形回归或圆形方差分析
